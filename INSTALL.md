# Install Guide

## Option 1: Manual install

1. Download or clone this repository
2. Copy the folders under `skills/` into your local Codex skills directory
3. Restart or refresh your Codex session if needed

## Option 2: Keep as a reference repo

If you do not want to copy the skills immediately, you can still keep this repo as a reference pack and manually reuse the prompts and structures inside each `SKILL.md`.

## Recommended Public Release Flow

1. Create a new GitHub repository
2. Upload the contents of this folder
3. Add a license
4. Add a short repository description
5. Mark which skills are stable and which are experimental
